Configuración de Git

Ahora que ya has abierto el Git Bash, te encontrarás con una consola color negro, no te asustes, esta ventanita te será muy útil de aquí en adelante ya que aquí harás la gran mayoría de los movimientos de tus archivos como te explicamos en Los Tres Estados.

Lo primero que deberás hacer cuando instales Git es establecer un nombre de usuario y un correo electrónico, ya que estos serán visibles cuando estés trabajando en conjunto con un equipo y se tenga un control de las modificaciones que se hagan a los archivos de un proyecto.

Lo primero que deberás hacer cuando instales Git es establecer un nombre de usuario y un correo electrónico, ya que estos serán visibles cuando estés trabajando en conjunto con un equipo y se tenga un control de las modificaciones que se hagan a los archivos de un proyecto.

                    $ git config --global user.name "Omar Pacheco"
                    $ git config --global user.email omarpacheco@example.com

Si quieres comprobar tu configuración utiliza este comando para mostrar todas las propiedades que Git ha configurado:

                    $ git config --list

¿Cómo obtener ayuda?

Si necesitas ayuda usando Git, existen tres formas de ver la página del manual para saber más acerca de cualquier comando en Git.

                    $ git help < verb>
                    $ git < verb> --help
                    $ man git-< verb>
