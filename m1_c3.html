Instalar Git y GitBash

Para empezar a utilizar Git, primero tienes que descargarlo e instalarlo en tu computadora, si ya lo tienes instalado, deberías intentar descargarlo de nuevo, ya que posiblemente ya haya una nueva versión. Si lo necesitas, puedes ver este video donde te explicamos paso a paso como instalar Git.

Para comenzar, haz click en el logo de Git que te dejamos debajo para descargarlo de su sitio Web.

Descargar Git

Dentro de su página te encontrarás con distintas opciones para su descarga, elige la que se adapte al sistema que tienes actualmente (32 o 64 bits), te recomendamos que elijas el tuyo de la sección Git for Windows Setup.

Al terminar de descargarse, quedará un archivo .exe, haz doble click sobre este y dale permisos para realizar cambios en tu computadora.

De aquí en adelante sigue una instalación típica de Windows, en la mayoría de los apartados dejaremos todo por defecto, hasta llegar a escoger nuestro editor de texto por defecto, ahi escogeremos el de nuestra preferencia (en nuestro caso utilizaremos Visual Studio Code).

Y de ahí en adelante dejaremos que el instalador haga el resto del trabajo, cuando terminemos, veremos en nuestra lista de programas una carpeta llamada Git, dentro de ella nos encontraremos con distintos programas, pero el que nos interesa para comenzar es Git Bash, ya que lo identifiquemos simplemente le daremos click para abrirlo.
